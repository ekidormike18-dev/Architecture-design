# Architecture Design Studio

A modern, premium architecture portfolio website showcasing architectural projects and designs.

## 📋 Project Overview

This project is a professional architecture design studio website built with HTML, CSS, and JavaScript. It features a sophisticated user interface with smooth interactions and a comprehensive portfolio of architectural projects.

## 🎯 Features

- **Responsive Design**: Modern, clean layout that works across different screen sizes
- **Smooth Navigation**: Smooth scroll functionality for seamless page navigation
- **Search Functionality**: Integrated search bar with real-time feedback
- **Project Showcase**: Display of 6 featured architectural projects with details
- **Add to Cart**: Interactive shopping cart functionality for project inquiries
- **Professional Branding**: Custom SVG logo with gradient effects
- **Modern UI/UX**: Glassmorphism effects, smooth transitions, and hover animations
